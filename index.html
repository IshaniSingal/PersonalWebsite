Create Personal Website

1. Create GitHub account by signing up on GitHub
2. Click on Your Repository as shown below

3. Click on Repositorie and then click on New
4. Enter the desired name "Name" for your website
5. Scroll down and click on Create repository.
6. Click on uploading an exosting file in blue
7. Drag and drop the HTML file along with any image file used in the html code to the repository
8. Click on Commit changes
9. Type "Name.github.io" on a new tab in web browser to access your website.
